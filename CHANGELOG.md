# Changelog

All notable changes to ChatGPTBox will be documented in this file.

## [2.5.9] - 2025-11-11

### Added
- API mode custom model configuration with support for custom names, model IDs, and display names
- Hide sidebar functionality in chat windows for better space management
- Enhanced focus mode with collapsible sidebars

### Changed
- Removed default custom model options from general API mode configuration
- Streamlined configuration panel interface for improved usability
- Improved API mode stability and reliability

### Fixed
- Enhanced configuration persistence
- Optimized sidebar toggle performance
- Fixed minor UI inconsistencies

### Technical
- Updated build process for better cross-platform compatibility
- Improved localization support for new features
- Enhanced manifest configurations for both V2 and V3
